Loading Data

To start working with the tool, use the Open File button from the File menu or the Tool Bar. Use the dialog window that appears to select a file to open.
The following file types are currently supported:

  • European Data Format (.edf)
  • Excel Worksheet (.xlsx, .xls)
  • Arrow IPC / Feather v2 (.feather)
  • Plain Text (.csv, .txt, .tsv)

After opening a file, you need to set the signal data source (i.e. the column / channel / etc in the file that contains the signal values) as well as the sampling rate of the signal.
A supplementary data source that should be included when computing results (e.g. temperature or O2-saturation recordings) can also be specified via the "Additional Data" field.
Once valid values have been set, pressing the Load Data button will read the specified data from the file and load it into the tool. A table of the loaded data will be displayed in the "Data View" section of the Input Data page.
