Getting Started

Data Import

File Selection

Press the Open File button to select a data file to open. Currently supported file types are:

  • Text files (.csv, .txt, .tsv)
  • Excel files (.xls, .xlsx)
  • Arrow IPC / Feather files (.feather, .ipc, .arrow)
  • EDF files (.edf)
If opening is successful, an attempt is made to automatically determine the sampling rate and signal data source; if this fails, the user will be prompted to manually set these parameters.
An additional data source (e.g. temperature or O2-saturation) can be specified in the "Additional Data" field. If specified, this data will be included when computing results and can optionally be displayed alongside the signal data in the main plot window.

Loading Data

Once the sampling rate, signal data and (optionally) additional data fields have been specified, press the Load Data button to load the data into the application.
If successful, a preview of the data will be displayed to the right of the import section in form of a table.
