Section List

The `Section List` window provides an overview of available sections and allows for quick navigation between them.

Components

Active Section

A text label that displays the name of the currently active section.

Command Bar

Holds commands for:

  • `New Section`: Only available when the base section (Section 000) is selected. A new section is created by visually selecting a range of the base section and clicking `Confirm`.
  • `Remove Section`: Can't be used when the base section is selected. Removes the currently active section (along with any associated data) from the list.
  • `Mark as Done`: Computes results for the current section and locks it to prevent further edits. See View and Export Results for more information.
  • `Unlock Section`: Removes the editing lock from the current section, so that it can be edited again.
  • `Show Section Summary`: Shows a dialog window with a summary of the current section's data.

Section List

The list of available sections. Clicking on a section name will select it as the active section. The active section is highlighted. Right-clicking a section will open a context menu with available commands for that section.
